## DETAILED SECTION SPECIFICATIONS

### HEADER SECTION (36" x 6")
**Font Sizes:**
- Title: 72pt Bold
- Subtitle: 36pt Regular  
- Authors: 28pt Regular
- Institution: 24pt Regular

**Content:**
```
OFFICE APOCALYPSE ALGORITHM
Predicting NYC Office Building Vacancy Risk Using Machine Learning

[Your Name] • Dr. Krishna Bathula (Faculty Advisor)
PACE University - Seidenberg School of Computer Science & Information Systems
```

### MAIN CONTENT GRID (6 sections, each ~10" x 15")

#### SECTION 1: ABSTRACT/INTRODUCTION (Top Left)
**Font Size:** 20pt body, 24pt headers
**Key Elements:**
- Problem statement (3-4 bullet points)
- Research objectives  
- Innovation highlights
- Significance metrics

#### SECTION 2: METHODOLOGY (Top Center) 
**Font Size:** 20pt body, 24pt headers
**Visual Elements:**
- Data leakage detection flowchart
- Model pipeline diagram
- Validation framework illustration

#### SECTION 3: RESULTS OVERVIEW (Top Right)
**Font Size:** 20pt body, 24pt headers
**Key Metrics (Large, Bold):**
- **92.41% ROC-AUC**
- **93.01% Precision@10%**
- **3.1x Efficiency**

#### SECTION 4: DATASET/PREPROCESSING (Bottom Left)
**Font Size:** 20pt body, 24pt headers
**Visual Elements:**
- Data sources infographic
- Feature engineering pipeline
- Building count by borough

#### SECTION 5: DETAILED RESULTS (Bottom Center)
**Font Size:** 18pt body, 22pt headers
**Tables & Charts:**
- Model comparison table
- SHAP feature importance
- Geographic risk heatmap

#### SECTION 6: CONCLUSIONS (Bottom Right)  
**Font Size:** 20pt body, 24pt headers
**Content:**
- Key contributions (numbered list)
- Research impact
- Future directions

### FOOTER SECTION (36" x 6")
**Font Size:** 18pt
**Layout:** 4 columns
- References (8")
- Acknowledgements (8") 
- Contact Info (8")
- QR Code to Dashboard (8")

## CONTENT DENSITY GUIDELINES

**For 36"x48" readability:**
- Maximum 150-200 words per section
- Bullet points over paragraphs
- 2-3 key visuals per section
- White space for visual breathing room
- High contrast between text and background

**Viewing Distance Optimization:**
- Readable from 3-6 feet away
- Key metrics visible from 10+ feet
- Clear visual hierarchy guides eye flow
